Detailed Description

template<class InterfaceClass>
class ibase::TFactorisableContainer< InterfaceClass >

Specific container implementation for factorisable items.

This container implementation supports folowing features:

  • Automatic memory management for created items
  • Serialization mechanism with automatic item allocation.
