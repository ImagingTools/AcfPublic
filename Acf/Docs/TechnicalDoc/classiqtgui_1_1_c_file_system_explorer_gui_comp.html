Detailed Description

UI component for visualization of file system contents.

CFileSystemExplorerGuiComp is implemented as an observer of IFileNameParam. Connected to model, this component will show the file system item, provided by the data model. Model data will be changed, if the the user selects a supported file item in the file system view.
