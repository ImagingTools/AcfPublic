Detailed Description

template<class SupplierInterface, class Product>
class iproc::TSupplierCompWrap< SupplierInterface, Product >

Wrapper implementation of interface iproc::ISupplier with preparation for component implementation.

During component initalization you should call AddInputSupplier for all suppliers used by this component as a input.
